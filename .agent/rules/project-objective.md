# Objectif du Projet : My Book Box App

## Vision
Créer une application sociale et collaborative permettant aux utilisateurs de localiser, explorer et contribuer aux "Boîtes à Livres" (Book Boxes) dans leur quartier.

## Fonctionnalités Clés
1.  **Cartographie Interactive** : Visualiser les boîtes à livres sur une carte (Leaflet) pour les trouver facilement.
2.  **Exploration du Contenu** : Voir quels livres sont disponibles dans une boîte spécifique (titre, auteur, thumbnail).
3.  **Contribution** :
    - Ajouter des livres (via scan ISBN ou recherche Google Books).
    - Signaler une boîte.
4.  **Monétisation / Sponsoring** : Système de livres sponsorisés (livres mis en avant dans les listes).

## Cible Utilisateur
- Amateurs de lecture.
- Personnes souhaitant donner une seconde vie à leurs livres.
- Promeneurs urbains.
